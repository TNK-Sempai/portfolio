01 / Le projet

Un bot qui apprend.

TNK Adaptive est un bot de trading algorithmique open source qui surveille en temps réel le top 50 paires USDT par volume sur Bybit, détecte des signaux d'entrée via RSI + Moving Averages, et apprend automatiquement de ses propres performances.

02 / Fonctionnalités

Ce que ça fait.

👁

SURVEILLANCE

Top 50 paires USDT, mis à jour toutes les 5 minutes. Suivi en temps réel des volumes et des mouvements de marché.

🎯

STRATÉGIE

RSI(14) + MA croisement. Long/Short automatiques. Stop Loss -3%, Take Profit +6%.

🧠

APPRENTISSAGE

Après 10 trades par paire, ajuste RSI, SL et TP selon win rate. Le bot s'améliore en continu.

📊

DASHBOARD

Chandelles TradingView en temps réel, positions live, historique complet des trades.

03 / Moteur adaptatif

Comment il apprend.

Condition Ajustement automatique
Win rate < 40% RSI rendu plus restrictif
SL touché > 60% Stop Loss élargi
TP touché > 60% Take Profit augmenté
PnL moyen négatif Ratio R:R corrigé
Win rate > 65% Paramètres assouplis

05 / Installation rapide

Démarrer en 60 secondes.

bash 
git clone https://github.com/TNK-Sempai/TNK-ADAPTIVE.git
cd TNK-ADAPTIVE
python -m pip install -r requirements.txt
python main.py
# Dashboard → http://localhost:5000

✓  Aucune clé API requise. ✓  Aucun risque financier.
